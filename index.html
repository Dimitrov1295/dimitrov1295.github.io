Image to ASCII

This app takes an image, goes through each pixel and calculates it's brightness. For each brightness, it inserts a character that has a "brightness" (the fatter the character, the brighter it is for example: "&" brighter than " ") similar to that pixel's brightness. After that is done, the resulting text looks like the source image.
